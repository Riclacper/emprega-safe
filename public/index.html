Sobre o EmpregaSafe

Objetivo

O sistema analisa vagas de emprego com base em regras de risco para indicar sinais de fraude, cobrança indevida e exposição desnecessária de dados pessoais.

Stack

Node.js, Express, HTML, CSS e JavaScript puro. A persistência está em arquivos JSON para facilitar demonstração, testes e evolução posterior.

Evolução sugerida

Você pode migrar a persistência para MongoDB, adicionar autenticação, perfis de usuário, dashboards avançados e modelos de IA para classificação mais robusta.
